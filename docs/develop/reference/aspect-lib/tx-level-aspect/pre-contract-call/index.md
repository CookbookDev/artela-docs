# PreContractCallCtx

>

## Where was it created

At [evm.go](https://github.com/artela-network/artela-evm/blob/4f1dd63feab4242118e331f677d4e05c267ea38e/vm/evm.go#L293),
before evm call to create `PreContractCallCtx` ，and execute Join point `PreContractCall`.

<!-- @formatter:off -->
```javascript
    //create request
    ...
    txAspect := &types.EthTxAspect{
        Tx:          tx,
        CurrInnerTx: inner,
        GasInfo:     &types.GasInfo{Gas: gas},
    }
    aspectRes := djpm.AspectInstance().PreContractCall(txAspect)
    if hasErr, err := aspectRes.HasErr(); hasErr {
        return nil, aspectRes.GasInfo.Gas, err
    }
    ...
 ```
<!-- @formatter:on -->

## Use Cases


### Transaction Context

#### Get the currently executed transaction

* return
    * [EthTransaction](/docs/classes/proto.EthTransaction.html)

```typescript
    let transaction = ctx.tx.content.unwrap()!
```

#### Get Current inner Call

* return
  * [EthStackTransaction](/docs/classes/proto.EthStackTransaction.html)

```typescript
    let call = ctx.currentCall;
```

### Trace Context
> Aspects can trace the changes of a Smart Contract's state，however this tracing is facilitated by extra opcodes and IR methods generated by the ASOLC compiler。
> TraceContext see [details](/develop/reference/aspect-lib/) on how to use it。

 
#### Get Call Tree

* return
  * [EthCallStacks](/docs/classes/proto.EthCallStacks.html)

```typescript
    let ethCallStacks = ctx.trace.callTree();
```

#### Find Call by Index

* return
  * [EthStackTransaction](/docs/classes/proto.EthStackTransaction.html) | null

```typescript
    let traceContext = ctx.trace.findCall(1);
```

#### Find Call by Index
* Parameter
  * index: tree index ,start with 1
* return
  * [EthStackTransaction](/docs/classes/proto.EthStackTransaction.html) | null

```typescript
    let traceContext = ctx.trace.findCall(1);
```

#### Query State Changes Indices
* Parameter
  * addr: account hex string, like `0xxabcccxxxddeddd`
  * variable: string ，like 'HoneyPot.balances'
  * indices: Array<Uint8Array\>  optional
* return
  * [EthStateChangeIndices](/docs/classes/proto.EthStateChangeIndices.html) | null

```typescript
    let indices = ctx.trace.stateChangeIndices("0xaaabbbccc",'HoneyPot.balances',[]);
```


#### Query State Changes

  * [EthStateChanges](/docs/classes/proto.EthStateChanges.html) | null

### StateDB

#### Get Balance

> retrieves the balance from the given address or 0 if object not found

* Parameter
    * address: account address
* return
    * (string): balance value, big int string

```typescript
    let balance = ctx.stateDB.balance("0x111222333444555666");
```

#### Get Nonce

> returns the nonce of account, 0 if not exists.

* Parameter
    * address: account address
* return
    * (i64): nonce value

```typescript
     let nonce = ctx.stateDB.nonce("0x111222333444555666");
```

#### Get State

> retrieves a value from the given account's storage trie.

* Parameter
    * address: account address
    * hash:  one key, hash hex string
* return
    * (string): state , hash hex string

```typescript
     let state = ctx.stateDB.stateAt("0x111222333444555666", "0x9999988888xxx");
```

#### Get Refund

> returns the current value of the refund counter.

* return
    * (i64): the current value of the refund counter

```typescript
    let refund = ctx.stateDB.refund();
```

#### Get CodeHash

> returns the code hash of account.

* Parameter
  * addr: address hash hex string
* return
    * (i64): the current value of the refund counter

```typescript
    let codeHash = ctx.stateDB.codeHash("0x111222333444555666");
```

### Evm Call

#### JitCall

>Executes a new message call immediately, without creating a transaction on the block chain.

* Parameter
    *  [JitInherentRequest](/docs/classes/proto.JitInherentRequest.html)
* Return
    *  [JitInherentResponse](/docs/classes/proto.JitInherentResponse.html)

```typescript
   let request = new JitInherentRequest(
          sys.utils.hexToUint8Array(walletAddress),
          new Uint8Array(0),
          new Uint8Array(0),
          sys.utils.hexToUint8Array(callData),
          sys.utils.hexToUint8Array(ethereum.Number.fromU64(1000000).encodeHex()),
          sys.utils.hexToUint8Array(ethereum.Number.fromU64(1000000).encodeHex()),
          new Uint8Array(0),
          new Uint8Array(0),
          new Uint8Array(0),
  );
  let response = ctx.jitCall.submit(request);
```
